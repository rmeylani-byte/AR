# GitHub Pages AR Setup

## Files included
- marker-test.html — test with red cube
- marker-model.html — uses your model.glb from GitHub

## How to publish
1. Go to https://github.com/rmeylani-byte/AR
2. Click Settings → Pages
3. Under “Build and deployment” → Source: choose “Deploy from a branch”
   Branch: main | Folder: /(root)
4. Click Save and wait 1–2 minutes.
5. Visit:
   - https://rmeylani-byte.github.io/AR/marker-test.html
   - https://rmeylani-byte.github.io/AR/marker-model.html
